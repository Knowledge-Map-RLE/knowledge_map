# Проект: Knowledge Map

## Обзор
Knowledge Map — это веб-приложение для визуализации и управления графом знаний. Система позволяет создавать блоки информации, связывать их между собой и автоматически размещать на карте с помощью алгоритма укладки.

## Архитектура
Проект состоит из нескольких микросервисов:

### 1. Frontend (React + TypeScript + PIXI.js)
- **Локация**: `client/`
- **Порт**: 5173 (dev server)
- **Технологии**: React, TypeScript, PIXI.js для рендеринга, Vite

### 2. API сервер (FastAPI + Neo4j)
- **Локация**: `api/`
- **Порт**: 8000
- **База данных**: Neo4j
- **Технологии**: FastAPI, neomodel, strawberry-graphql

### 3. Сервис укладки (Python + gRPC)
- **Локация**: `layering/`
- **Технологии**: Python, gRPC, алгоритмы топологической сортировки

## Основные сущности
- **Block**: Узел графа с контентом, координатами и метаданными
- **Link**: Направленная связь между блоками
- **Level**: Уровень в иерархии (горизонтальные слои)
- **Sublevel**: Подуровень для группировки блоков

## Текущая проблема
В клиенте операции создания связей, удаления блоков и связей выполняются только локально, не сохраняясь в базе данных. Требуется интеграция с серверным API.
