# Извлечение лингвистических свойств. Промпт.

Для приложенного текста определи по очереди, лингвистические свойства всего текста, каждого предложения, каждого слова, обознач ошибки если есть. Если из предложенных вариантов каждой структуры, каждого уровня, не подходит вариант из списка или свойство не удалось определить, нужно присвоить ему значение 'Другой вариант'.

- Осложнённость

## Пайплайн

- Получить словарь из текстов.
- Сделать аннотацию каждого слова по лингвистическим свойствам указанным выше (классифицировать по грамемам).
- Вытащить паттерны.
- Разметить предложения. (написать для этого промпт)
- Проверить текст на ошибки (левенштейн) и исправить их. (движок правописания)
- Посчитать какие паттерны способствуют более длинной цепочке действий и достижению конкретной цели
- Ещё нужен "словарь" структур которые могут встречаться в тексте: формулы, числа и их единицы измерения, таблицы (должны быть в Markdown-файле написаны на HTML), и так далее. Словарь скорее всего должен быть из регулярок.
- Проверить покрытие структур языка.
