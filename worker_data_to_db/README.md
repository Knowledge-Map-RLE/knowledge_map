# Загрузка и сохранение данных в базу данных

## Доступные воркеры

- PubMed baseline FTP (загрузчик статей, сохранение в Neo4j базу данных)

## Локальный запуск (Poetry)

1) Требования: установлен Poetry, запущен Neo4j (`bolt://127.0.0.1:7687`).

2) Установка зависимостей:

```
poetry install
```

3) Загрузка одного XML.gz файла (предпоследнего в каталоге `D:/Данные/PubMed`):

```
poetry run python pubmed_baseline_ftp_to_db.py
```

4) Проверка статистики в БД:

```
poetry run python analyze_database.py
```

Примечание: скрипт загрузки временно настроен на обработку одного файла и выполняет полную очистку БД через APOC перед загрузкой.
