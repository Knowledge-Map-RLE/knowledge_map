# Ядро Карты Знаний

Это основа Карты Знаний наравне со специализированной укладкой, микросервис, у которого есть единственная задача — максимально быстро достич на практике радикального продления жизни (РПЖ) для человека. Ускорять науку в общем и достигать конкретных целей в частном. Отвечает на вопрос: "Что делать для практического достижения РПЖ?".
